13 //Rijndael (pronounced Reindaal) is a block cipher, designed by Joan Daemen and Vincent Rijmen as a candidate algorithm for the AES.
14 //The cipher has a variable block length and key length. The authors currently specify how to use keys with a length
15 //of 128, 192, or 256 bits to encrypt blocks with al length of 128, 192 or 256 bits (all nine combinations of
16 //key length and block length are possible). Both block length and key length can be extended very easily to
17 // multiples of 32 bits.
18 //Rijndael can be implemented very efficiently on a wide range of processors and in hardware.
19 //This implementation is based on the Java Implementation used with the Cryptix toolkit found at:
20 //http://www.esat.kuleuven.ac.be/~rijmen/rijndael/rijndael.zip
21 //Java code authors: Raif S. Naffah, Paulo S. L. M. Barreto

28  //The Electronic Code Book (ECB), Cipher Block Chaining (CBC) and Cipher Feedback Block (CFB) modes

30  //In ECB mode if the same block is encrypted twice with the same key, the resulting
31  //ciphertext blocks are the same.
32  //In CBC Mode a ciphertext block is obtained by first xoring the
33  //plaintext block with the previous ciphertext block, and encrypting the resulting value.
34  //In CFB mode a ciphertext block is obtained by encrypting the previous ciphertext block
